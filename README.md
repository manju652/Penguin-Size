❓ Problem Statement

To classify penguin species based on physical measurements using machine learning algorithms and compare their performance.

---

📌 Project Overview

This project applies multiple classification algorithms on the Penguin dataset to analyze and predict species based on features like flipper length, body mass, and bill dimensions.

---

🚀 Algorithms Implemented

- 🔹 K-Nearest Neighbors (KNN)
- 🔹 Support Vector Machine (SVM)
- 🔹 Decision Tree

---

📊 Dataset Features

- Flipper Length
- Body Mass
- Bill Length
- Bill Depth

---

🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
